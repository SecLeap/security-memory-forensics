# 网络与协议

## 学习目标

只学习内存镜像中可见的网络状态：连接、socket、DNS 缓存、接口与缓冲区；不将本专题扩展为网络流量分析或安全运营。

## 推荐资料

- [Volatility 3 Linux 教程](https://volatility3.readthedocs.io/en/latest/getting-started-linux-tutorial.html)：含 Linux 网络对象插件示例。
- [Volatility 3 Windows 教程](https://volatility3.readthedocs.io/en/latest/getting-started-windows-tutorial.html)：从平台插件能力了解 Windows 镜像分析入口。
