# 取证基础

## 学习目标

掌握内存镜像的易失性、采集扰动与结论边界。内存取证的每项发现都应能回答：来自哪个镜像、由哪个工具/版本/参数得到、能否独立复核、有哪些替代解释。

## 推荐资料

- [NIST SP 800-86](https://csrc.nist.gov/pubs/sp/800/86/final)：数字取证技术整合的通用方法论。
- [Volatility 3 输出渲染器文档](https://volatility3.readthedocs.io/en/latest/renderers.html)：理解如何保留结构化、可复核的原始输出。

## 最小实验

建立一份镜像登记表：镜像编号、目标系统与版本、采集工具及哈希、采集开始/结束时间、采集操作者、存放位置、分析工具版本。
