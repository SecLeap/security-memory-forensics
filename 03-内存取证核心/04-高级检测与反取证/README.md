# 高级检测与反取证

## 进入条件

完成基础对象、符号和插件输出的交叉验证后，再学习本专题。高级分析的核心不是寻找“一个神奇插件”，而是比较不同枚举路径：链表遍历、对象扫描、页表/内存映射、模块加载器和原始字节是否给出一致结果。

## 推荐资料

- [Volatility 3 插件开发文档](https://volatility3.readthedocs.io/en/latest/writing-plugins.html)：理解插件能力与限制。
- [YARA 编写规则](https://yara.readthedocs.io/en/stable/writingrules.html)：只为经授权的教学镜像编写与验证规则。

## 实验产出

为每个实验异常点记录：异常对象地址、两种以上定位方法、原始字节/结构字段、正常对照对象、可能误报原因与最终结论。
