# 凭据与敏感数据驻留

## 学习资料与练习

- [Windows 令牌访问权限](https://learn.microsoft.com/windows/win32/secauthz/access-tokens)
- [Apple Platform Security](https://support.apple.com/guide/security/welcome/web)

练习只使用自行创建的测试账号、测试令牌和无敏感内容的样本。报告只记录对象类型、位置和验证结果，不记录、导出或传播真实凭据、密钥或个人数据。
