# 内存取证场景

本目录仅收录可通过终端内存镜像定位、验证和解释的现象；每个场景后续应包含前置条件、目标内存对象、分析步骤、交叉验证与结论边界。

## 场景练习方法

每个场景采用“正常对照镜像 → 单一已知异常 → 双重验证 → 书面结论”的顺序。练习时不需要真实攻击链：可用测试进程、公开教学镜像或刻意加载的无害模块构造可观察差异。
