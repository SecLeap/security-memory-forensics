# DumpIt 实战手册

## 获取与安装

DumpIt 属旧版 Windows 内存采集工具。当前截图中的版本和可靠官方发布来源无法确认时，**不建议新装**。如有已授权的 Comae/内部归档版本，必须核验数字签名、哈希、许可证和对目标 Win11 的兼容性；否则使用 [WinPmem](14-WinPmem.md) 或经验证的替代采集方案。

## 验证与最小使用

仅在干净、隔离 Win11 快照上测试采集，记录工具版本、驱动行为、输出格式、镜像 SHA-256 和 Volatility 3 解析结果。采集工具可能改变内存，故采集日志是证据的一部分。

## 内存取证联动

先验证镜像类型和 `windows.info`，再开展对象分析。进程 dump、内核转储和物理 RAM 镜像覆盖范围不同，不能相互替代。

## 使用方法

1. 仅在旧版工具来源、签名和 Win11 兼容性已验证的干净快照测试；否则不运行。
2. 记录采集确认提示、版本、输出格式、时间、大小和 SHA-256。
3. 用 Volatility 3 验证解析能力，并与 WinPmem 受控采集结果比较覆盖范围。

## 实战场景与完成标准

场景：历史镜像格式的兼容性评估。完成标准是明确工具版本与系统限制，不将测试签名/安全机制变更作为可接受前提。
