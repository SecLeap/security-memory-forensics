# Windows 11 工具实战手册

每个工具一份实战手册。共同规则：仅从官方页面获取；先记录下载 URL、版本、SHA-256/签名与许可证；在隔离 Win11 虚拟机中准备工具并创建快照。不要使用来源不明的旧版、整合包或截图中的未知快捷方式。
