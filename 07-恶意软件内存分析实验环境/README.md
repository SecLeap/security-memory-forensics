# 恶意软件内存分析实验环境

本目录服务于终端内存取证学习：通过隔离的 Windows 11 分析机与 Ubuntu 模拟服务机，产生可控的进程、文件、注册表和网络行为，并在执行前后制作内存镜像进行关联分析。它不是生产终端处置手册，也不提供样本投递、持久化或规避方法。

单个工具的安装与使用以 [05-工具与自动化](../05-工具与自动化/README.md) 为唯一入口；本目录只维护网络隔离、虚拟机快照、服务组合与联合实验，不复制单工具安装步骤。

## 基本原则

- 仅使用自建无害测试程序、公开教学样本或已获得明确授权的材料。
- Win11 与 Ubuntu 仅接入隔离虚拟网络；禁用桥接、NAT、主机共享目录、拖放、共享剪贴板和 USB 直通。
- 每次实验前创建“干净快照”，实验完成后导出证据并还原快照；不要复用受污染虚拟机。
- 任何镜像、PCAP、Procmon 日志和模拟服务日志都可能含敏感数据，按证据规则保管。
