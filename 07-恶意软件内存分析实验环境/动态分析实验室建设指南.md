# 动态分析实验室建设指南（Malware Dynamic Analysis Lab）

## 1. 实验室目标

本实验室用于：

- 恶意软件动态分析
- 勒索软件分析
- 挖矿木马分析
- 远控木马分析（RAT）
- APT样本分析
- 内存取证
- IOC提取
- 应急响应溯源
- 攻击链还原
- YARA规则开发验证

# 5. 工具体系

## 5.1 Sysmon

### 作用

记录：

```text
进程创建
网络连接
文件创建
注册表修改
DLL加载
驱动加载
WMI
```

### 场景

发现：

```text
powershell
rundll32
regsvr32
wmic
mshta
```

等攻击行为。

---

### 安装

```cmd
Sysmon64.exe -accepteula -i sysmonconfig.xml
```

---

### 基础用法

查看日志：

```cmd
eventvwr.msc
```

路径：

```text
Applications and Services Logs
Microsoft
Windows
Sysmon
Operational
```

---

### 高级用法

#### 查找进程树

```powershell
Get-WinEvent `
-LogName "Microsoft-Windows-Sysmon/Operational"
```

#### IOC提取

关注：

```text
EventID 1
EventID 3
EventID 7
EventID 11
EventID 13
```

# 5.3 Noriben

## 作用

Procmon自动化分析框架。

自动：

```text

运行样本
生成报告
```

---

## 工作流程

```text
Procmon
     │
     ▼
Noriben
     │
     ▼
过滤系统噪声
     │
     ▼
行为报告
```

---

## 基础用法

```cmd

```

---

## 高级用法

自定义：

```text
Noriben.config
```

过滤：

```text
Windows Noise

Defender Noise
```

---

```text

```

# 6.2 Volatility3

## 作用

分析：

```text
进程
DLL
网络连接
恶意代码
```

---

## 基础用法

```bash

```

---

## 高级用法

### 进程树

```bash
vol -f memory.raw windows.pstree
```

### 命令行

```bash
vol -f memory.raw windows.cmdline
```

### 网络连接

```bash
vol -f memory.raw windows.netscan
```

### DLL

```bash
vol -f memory.raw windows.dlllist
```

### 注入检测

```bash
vol -f memory.raw windows.malfind
```

### YARA扫描

```bash
vol -f memory.raw windows.yarascan
```
