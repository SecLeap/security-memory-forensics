# 恶意软件样本仓库

本目录索引可用于授权研究、恶意软件分析和内存取证实验的公开样本与分析服务。样本、哈希、查询记录和分析结果均可能含有敏感案件线索；在访问或下载前，确认组织授权、适用法律和实验室隔离措施。

## 仓库与服务

| 名称 | 入口 | 主要用途 | 访问提示 |
| --- | --- | --- | --- |
| VirusTotal | [virustotal.com](https://www.virustotal.com) | 文件、URL、域名和 IP 的多引擎检测与关联情报查询 | 上传文件或提交哈希可能泄露样本存在性与案件线索；仅在授权范围内查询。 |
| Hybrid Analysis | [hybrid-analysis.com](https://hybrid-analysis.com/) | 在线沙箱行为分析与已有报告检索 | 提交样本会发送至第三方服务；避免提交内部、受限或包含敏感数据的文件。 |
| VirusShare | [virusshare.com](https://virusshare.com/) | 恶意软件样本集合 | 访问和样本下载通常受账户与使用条款约束；仅在完全隔离的研究环境中处理。 |
| MalwareBazaar | [bazaar.abuse.ch](https://bazaar.abuse.ch/) | abuse.ch 运营的恶意软件样本与元数据索引 | 样本仅用于经授权的研究或防御用途；先记录 SHA-256 和来源，再按实验记录流程保管。 |
| VX-Underground | [vx-underground.org](https://vx-underground.org) | 恶意软件研究资料、样本与档案索引 | 遵守站点条款和本地法律；不要将下载内容放入本仓库或非隔离设备。 |

## 使用边界

- 本 Git 仓库不提交恶意样本、压缩包、PCAP、内存镜像、可执行提取物或可直接执行的投递材料。
- 在实验前记录来源、访问日期、样本 SHA-256、授权依据和实验编号；样本保存在受控、加密且访问受限的位置。
- 仅在禁用桥接、NAT、共享目录、剪贴板、拖放和 USB 直通的隔离虚拟网络中处理样本，具体要求见[恶意软件内存分析实验环境](../../07-恶意软件内存分析实验环境/README.md)。
- 外部检测或沙箱结论只能作为情报线索；内存取证结论仍须由本地镜像中的独立证据交叉验证。

## 实验记录最小字段

| 字段 | 说明 |
| --- | --- |
| 实验编号 | 可追溯的内部实验或案例编号 |
| 来源与访问时间 | 站点名称、入口和访问时区 |
| 样本标识 | 优先 SHA-256；不在公开文档记录敏感文件名或内部路径 |
| 授权与用途 | 授权范围、研究/教学目的和限制 |
| 隔离快照 | 虚拟机与网络快照标识 |
| 证据位置 | 受控存储位置、镜像哈希与保管责任人 |
