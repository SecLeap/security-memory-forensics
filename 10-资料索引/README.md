# 资料索引

## 核心资料（优先官方）

- [Volatility 3 文档](https://volatility3.readthedocs.io/en/latest/) 与 [源码](https://github.com/volatilityfoundation/volatility3)
- [Windows Debugger / dump 文档](https://learn.microsoft.com/windows-hardware/drivers/debugger/)
- [Linux Kernel 文档](https://docs.kernel.org/) 与 [kdump](https://docs.kernel.org/admin-guide/kdump/kdump.html)
- [Apple XNU 源码](https://github.com/apple-oss-distributions/xnu) 与 [Apple Platform Security](https://support.apple.com/guide/security/welcome/web)
- [AVML](https://github.com/microsoft/avml)、[LiME](https://github.com/jtsylve/LiME)、[WinPmem](https://github.com/Velocidex/WinPmem)
- [YARA 文档](https://yara.readthedocs.io/)
- [NIST SP 800-86](https://csrc.nist.gov/pubs/sp/800/86/final)

## 资料使用规则

- 优先链接到官方文档、项目 release 或源码；二手文章只作为辅助说明。
- 为每条实践资料记录访问日期、对应工具/系统版本及是否已在实验机验证。
- 历史工具和旧案例必须标注适用系统年代，不能默认支持当前 Windows、Linux 或 macOS。
