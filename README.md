# 终端内存取证知识库

面向 Windows、Linux、macOS 终端内存取证的技术仓库。范围限定为内存镜像的采集、解析、分析、验证和取证交付；恶意软件分析环境仅用于生成受控内存取证证据，不扩展到网络安全运营、应急响应流程或通用攻击场景。

## 能力跃迁路线

```text
基础技术储备 → 平台内存机制 → 内存采集与解析 → 对象与异常分析
                                                   ↓
                                   内存取证场景 → 工具自动化 → 证据报告
```
